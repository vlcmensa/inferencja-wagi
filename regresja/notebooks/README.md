# Notebooks - Softmax Regression

Jupyter notebooks for training and experimenting with the softmax regression model.

## Files

| Notebook | Description |
|----------|-------------|
| `Siec_2_warstwy.ipynb` | Main training notebook for softmax regression on MNIST |

## Usage

Open notebooks using Jupyter:

```bash
jupyter notebook
```

Or use VS Code / Cursor with Jupyter extension.

## Purpose

- Train and evaluate softmax regression model
- Experiment with quantization parameters
- Visualize training results and accuracy
